# editor
HTML, CSS, JS code editor.
## Install and run
```sh
$ git clone https://github.com/vidushisinha/editor.git
```
```sh
$ npm install
```
```sh
$ npm run dev
```
## Features
-    HTML/CSS/JS code editing
-    Code formatting
-    Built-in react
-    Auto run
-    Saves in HTML format while download
